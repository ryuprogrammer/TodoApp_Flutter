## 概要
Supabase、Riverpod、freezedを使用してTodoアプリを作成。

## 使用技術
| 用途 | 技術 |
| ---- | ---- |
| 状態管理 | Riverpod, freezed |
| バックエンド | Supabase |
| その他 | riverpod_generator |
